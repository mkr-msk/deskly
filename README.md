# Deskly

Deskly — это простой веб-сервис для учёта и обработки рабочих заявок внутри небольшой команды.

Проект разрабатывается как портфолио с полным циклом разработки по Agile:
от формирования требований и документации до реализации, тестирования и деплоя.

## Основные возможности (MVP)
- регистрация и авторизация пользователей
- создание заявок
- просмотр списка заявок
- изменение статуса заявки
- комментарии к заявкам
- роли пользователей (User, Agent, Admin)

## Структура проекта

```
deskly/
├── docs/ # документация проекта (vision, backlog, архитектура и т.д.)
├── src/ # исходный код
└── README.md
```

## Технологии (план)
- Frontend: Next.js + TypeScript
- Backend: Next.js API
- Database: PostgreSQL
- ORM: Prisma
- CI/CD: GitHub Actions
- Hosting: Timeweb
